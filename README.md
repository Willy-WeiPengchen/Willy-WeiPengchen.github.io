本仓库为“链元体系”的基础开源部分，旨在促进行业交流与通用场景的生态共建。为明确权利边界，特作如下严正声明：

### 1. 代码层开放
本仓库提供的基础业务代码严格遵循 **Apache 2.0** 许可证，允许在任何领域（包括教育、商业等）进行自由使用、修改与分发，助力产学研协同共建。

### 2. 核心算法剥离（闭源区）
本仓库**不包含**“切分生成拓扑V2.0”等核心底层引擎算法。上述核心算法受独立知识产权与专利法严格保护，仅限通过官方白名单准入机制的合规共建企业获取授权。

### 5. 关于“链元”名称的公共属性声明
正如“词元（Token）”已成为全球AI行业的通用计量单位，“链元（Tokener）”自诞生之日起，即定位为AI 2.0时代价值计量的通用基础术语。发明人韦朋辰在此明确声明：**放弃将“链元”作为排他性商标进行注册与垄断**。
我们欢迎全行业在学术探讨、技术文档、开源共建中自由、规范地使用“链元”一词。真正的壁垒不在于名词的独占，而在于底层逻辑与核心算法的持续领先。
